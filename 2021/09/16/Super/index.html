在 Python 中定义子类时，super() 用于继承父类的属性或方法。super() 一般用于子类的 __init()__ 方法中，防止它将父类的 __init()__ 方法覆盖掉。

具体语法

  • type：该子类的名称；
  • object-or-type：一般情况下为 self；
  • method()：需要被继承的父类方法；
  • some-arguments：被继承的父类方法的参数，如果父类中该方法没有传入参数或传入参数有默认值，则此处可以省略（将会直接使用参数的默认值）；

注意：上述语法同时适用于 Python 2.x 与 Python 3.x，而对于 Python 3.x，该语法还可以直接省略为 super().method(some-arguments)；

主要用途

在定义子类时，如果给子类实现了 __init__() 方法，则默认它会覆盖掉父类的 __init__() 方法（导致父类中初始化定义的属性无法被子类访问），此时，可以使用 super() 将父类的 __init__() 也继承过来，使得子类可以同时访问两种 __init__() 方法中定义的属性：
